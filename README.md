# Undefined Relationship AI Companion Beta

Undefined Relationship is an early paid beta for a relationship-driven AI companion app.

It is not a character-preset toy, not human companionship, and not therapy. The product explores a more constrained question:

Can an AI companion feel more useful when the relationship grows through long-term interaction, memory, boundaries, proactive messages, and AI moments instead of being fully defined at the start?

## Current Beta Focus

- Relationship state that changes through interaction
- Long-term memory experiments
- Boundary-aware replies instead of unconditional pleasing
- Planned proactive messages and AI moments
- Lightweight operations flow for paid beta access
